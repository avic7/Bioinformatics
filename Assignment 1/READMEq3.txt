This python code reads a multi-FASTA file and summarizes its contents using the Biopython library. 
For each sequence in the file, it displays:
1. The header line.
2. The first 10 amino acids of the sequence.
3. The total number of amino acids in the sequence.

Libraries used : 
- Biopython library (install using `pip install biopython`)

How to run:
1. Save the script as `assq3.py`.
2. Place the multi-FASTA file in the same directory as the script.
3. Open VS code.
4. Run the script in treminal.
